# One Tap Memo

瞬発的な思考を途切れさせない、ラボテイストのワンタップメモです。Enterキーだけで保存でき、#タグの自動抽出やクイックタグ、検索・フィルタ、JSON/CSVでの入出力など、メモ管理に必要な道具をコンパクトにまとめました。

## セットアップ

```bash
npm install
npm run dev
```

- `npm run dev` : 開発サーバーを起動します。
- `npm run build` : 本番ビルドを出力します。
- `npm run preview` : ビルド成果物をローカルで確認します。
- `npm run test` : Vitest + Testing Library によるUIテストを実行します。

## 主な機能

- **Enterで即保存** : Shift+Enterで改行、Alt+Sで音声入力。
- **#タグ自動抽出** : 入力テキストからハッシュタグを検出し、検索やフィルタに活用。
- **クイックタグ** : 仕事/私事/アイデア/学び/ToDo をワンタップ付与。
- **検索・フィルタ** : キーワード、#タグ、クイックタグ、日付、ピン留めで絞り込み。
- **ピン・編集・コピー・削除** : 各メモカードに操作ボタンを常備。
- **JSON/CSV入出力** : ローカルストレージに保存されたメモをエクスポート/インポート。
- **テーマ切替** : 白×濃紫を基調としたライト/ダークテーマをワンタップで変更。

## デザインポリシー

- Zen Kaku Gothic New × Shippori Mincho × Inter を組み合わせた軽やかなタイポグラフィ。
- 物理感のある「ふせん」カードと柔らかなシャドウで、思考の手触りを演出。
- 最小限の遷移と待ち時間で「思考の速度を損なわない」操作感。
